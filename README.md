# Scripts

This repository is a collection of various small scripts, utilities and
configurations I use periodically.

- [Python utilities](./python/common/) usefull files for writing terminal python
  CLI applications
- [Bank statement parser](./python/bank_statement_parser.py) turns my PDF bank
  statements into easier to use CSV data
- [Image metadata printer](./python/img_metadata.py) prints image metadata
- [ANSI printer](./bash/ansi) quickly print ansi codes to terminal
- [CMD Reminder](./bash/cmds) used to remind myself of some temrminal commands.
- [Audiotest](./bash/audiotest) plays an audio test sound on key press (CAPS
  LOCK or NUM LOCK)
- [texcleanup](./bash/texcleanup) remove tex build files
